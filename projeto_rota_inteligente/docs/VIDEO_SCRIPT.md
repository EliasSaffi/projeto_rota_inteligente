
ROTEIRO DE VÍDEO (4 MINUTOS)

1. Introdução (30s)
Meu projeto se chama Rota Inteligente e tem como objetivo otimizar rotas de entrega utilizando algoritmos de inteligência artificial.

2. Problema (40s)
Empresas de delivery enfrentam dificuldades para planejar rotas durante horários de pico. Muitas vezes os trajetos são definidos manualmente, o que gera atrasos e aumento de custos.

3. Solução proposta (1min)
A solução desenvolvida representa a cidade como um grafo.
Cada ponto de entrega é um nó e as ruas são arestas.

Utilizamos o algoritmo A* para encontrar o menor caminho entre pontos.

Quando existem muitos pedidos, aplicamos K-Means para agrupar entregas próximas.

4. Demonstração do projeto (1min30)
Mostrar:
- diagrama do grafo gerado
- clusters de entregas
- mapa com pontos de entrega

Explicar rapidamente como o código executa essas etapas.

5. Conclusão (20s)
Esse projeto demonstra como conceitos fundamentais de IA podem ser aplicados para resolver problemas logísticos reais.
